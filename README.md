# Python-Mini-Projects
Short projects made in Python to streamline monotonous tasks, or simply as solutions to problems I find interesting.

## Overviews
### Determinants
 - This program finds the determinant of any matrix via two different methods.
 - I made this while bored on a plane flight.
 - In the time I would've been watching Linear Algebra lectures, I decided to take a break and create this instead - hey at least it's still related to the content.

### Polynomial Rational Roots
 - A short program made to find rational roots of a given polynomial.
 - Made on a whim when I was learning about rational root theorem in VCE methods.

### Polynomial Root Finder
 - This program approximates the roots of any degree polynomial - an extension from just rational roots.
 - This was an interesting problem that me and my friend Sean designed the algorithm for on a bit of a whim.

### Prime Sieves
 - A program made to test out the effiencies of two different prime-finding functions.
 - I was interested in prime numbers throughout Year 10 of high school, and that year would've been when I made this.

### Stackable
 - A program to investigate number 'stackability'.
 - This was made as I was curious how we could find stackable numbers, and how they're composed, based on a VCE Specialist question asking for a proof that no power of 2 is stackable.
 - See the actual program for more info.

### ElSchool Awards
 - My friends and I have a yearly 'awards night' where, among other events, we have an awards presentation.
 - We all vote on a series of questions, picking who in the friend group we think best fits the question's description.
 - Me and my friend Rune then put together a slideshow to present these results.
 - This program streamlines the processing of this data, quickly creating a base slideshow with the key results for us to build on.

### Double Sided Print
 - I typically print out lecture notes and practice exams so I can physically write on them.
 - My printer only supports single-sided printing, and this program helps me print out medium-large documents.
 - Importantly, it splits the printing load into groups of a small enough size such that I can easily staple the group together.

### Spotify Purger
 - A version of the program I made to try and filter down the 800 or so songs in my primary playlist on Spotify.
 - It would constantly run, and track whether I had skipped a song while listening to the playlist.
 - If I had skipped a song, it would remove the song from the playlist - essentially weeding out the good and the bad.

### Youtube Adblock
 - A short program I made after I my adblock extension started to break - it wasn't blocking youtube ads.
 - I noticed that youtube videos embedded in websites don't show ads.
 - So this program creates a simple HTML file with the given youtube video embedded in it; so no ads!
 - This exploit has since been addressed by youtube, I believe.
